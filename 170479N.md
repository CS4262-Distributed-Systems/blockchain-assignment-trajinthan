# Blockchain Assignment (CS4262-Distributed Systems)

## Concepts learnt about blockchains

Concept 1 :  Smart Contract - Building blocks of block chain based applications which contains business logic and computation. This allows the user to set up pre-determined rules for automated execution of code as well as automate a workflow. Generally smart contract is used in creating assets, reading assets and updating assets.

Concept 2 :  Consensus - Consensus helps to synchronie the state of ledgers throughout the blockchain. Consensus algorithm is used in this assignment to decide and agree with the order of transactions and ensure the state of ledgers.
